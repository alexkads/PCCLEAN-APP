🐛 SOLUÇÃO RÁPIDA DE PROBLEMAS
═════════════════════════════

"PCClean não pode ser aberto porque é de desenvolvedor não identificado"
  → xattr -cr /Applications/PCClean.app
  → Ou: System Preferences > Security > "Open Anyway"

"DMG não monta"
  → hdiutil verify build/PCClean-2.0.0.dmg
  → hdiutil attach -mountpoint /tmp/test build/*.dmg

"Script não executa"
  → chmod +x macos-installer/*.sh
  → Verificar que está no diretório correto

"Erro de compilação"
  → cargo clean
  → cargo build --release
  → cd macos-installer && ./build-installer.sh
